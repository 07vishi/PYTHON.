MAIN COMPONENTS OF A COMPUTER

  • 1. CPU (Central Processing Unit)

    The CPU is often referred to as the "brain" of the computer. It performs all the logical and arithmetic operations required to execute instructions from programs. It processes data and controls the flow of information within the system.

  • 2. Memory

    Memory, often referred to as RAM (Random Access Memory), temporarily stores data and instructions that the CPU needs while performing tasks. It is volatile, meaning it loses its contents when the power is turned off.

  • 3. Control Unit

    The Control Unit (CU) is a component of the CPU that directs the operation of the processor. It tells the memory, ALU, and I/O devices how to respond to the instructions that have been sent to the processor.

  • 4. Arithmetic Logic Unit (ALU)

    The ALU is another component of the CPU that performs all arithmetic and logical operations. This includes addition, subtraction, and comparisons (like greater than or less than).

  • 5. I/O Devices (Input/Output Devices)

    I/O devices allow the computer to interact with the outside world. Input devices (like keyboards and mice) allow users to input data, while output devices (like monitors and printers) display or produce results.

Example: Checking CPU Usage in Python


import psutil

# Check CPU usage
cpu_usage = psutil.cpu_percent()
print(f"Uso de CPU: {cpu_usage}%")

# Check available memory
available_memory = psutil.virtual_memory().available / (1024 * 1024)  # Convert bytes to MB
print(f"Memória disponível: {available_memory:.2f} MB")

Explanation of the Example:

The script imports the psutil library, which provides an interface for retrieving information on system utilization (CPU, memory, disks, network, sensors).

It also retrieves the available memory in bytes using psutil.virtual_memory().available, converts it to megabytes (MB), and prints it.

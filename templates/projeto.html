1. Visão Geral e Objetivos

O nosso projeto é uma plataforma inovadora que integra os serviços de saúde mental com um sistema de doação alimentar, utilizando uma combinação de tecnologias avançadas: Java, Python e C++.

O objetivo principal é criar um ecossistema digital que promova o acesso à saúde e combata a insegurança alimentar, alinhando-se com múltiplos Objetivos de Desenvolvimento Sustentável (ODS) da ONU.

Durante o cadastro inicial, os doentes terão a opção de indicar se eles requerem assistência alimentar. Esta informação será enviada para o microserviço C++ (gerenciado por Vishwa) que processará pedidos de doação.

Objetivos Específicos:

  • Facilitar o acesso a consultas médicas e orientação em saúde mental
  • Criar um sistema eficiente de doação e distribuição de alimentos
  • Promover a solidariedade e o envolvimento da comunidade
  • Desenvolver uma solução tecnológica escalável e segura

2. Alinhamento com ODS da ONU

ODS 2: Fome Zero e Agricultura Sustentável: Sistema de doação de alimentos
ODS 3: Saúde e Bem-Estar: Acesso facilitado aos profissionais de saúde
ODS 10: Redução das Desigualdades: Democratização do acesso à saúde e alimentação
ODS 17: Parcerias para os Objetivos: Colaboração entre setores da sociedade

3. Desafios Técnicos e Soluções

Integração de Linguagens: Uso de microserviços e APIs bem definidas
Escalabilidade: Arquitetura em nuvem com balanceamento de carga
UX/UI: Design responsivo e acessível para múltiplos dispositivos
Verificação de Usuários: Sistema robusto de verificação para profissionais de saúde

4. Considerações Éticas e Legais

Conformidade com regulamentações de saúde
Proteção de dados pessoais e médicos
Transparência no uso e distribuição das doações
Políticas de uso e termos de serviço claros

5. Distribuição de Tarefas por Membro da Equipe

5.1 Sofia (Java)

  • Responsabilidade principal: Gerenciar o sistema de consultas psicológicas
  • Implementar agendamento de consultas
  • Criar canais de comunicação entre pacientes e psicólogos
  • Integrar visualização de informações sobre doações de alimentos

5.2 Lara (Python/Flask)

  • Responsabilidade principal: Desenvolver interface e fluxo de registro
  • Implementar registro para todos os tipos de usuários
  • Conectar com API do Vishwa para gestão de doações
  • Criar as interfaces
  • Garantir fluxo de autenticação seguro

5.3 Vishwa (C++)

  • Responsabilidade principal: Implementar sistema de doação de alimentos
  • Desenvolver lógica de gestão de doações
  • Criar API para comunicação entre microserviços
  • Implementar lógica de solicitação de alimentos e relatórios
